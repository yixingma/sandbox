# Spring Boot Application

A bare minimum Spring Boot application with a simple REST API.

## Prerequisites

- Java 17 or later
- Maven 3.6+

## Build

```bash
mvn clean package
```

## Run

```bash
mvn spring-boot:run
```

The application will start on `http://localhost:8080`

## Testing

Access the endpoint:

```bash
curl http://localhost:8080/
```

Expected response: `Hello, Spring Boot!`
